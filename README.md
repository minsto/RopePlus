## Build / Compilation

This mod is made for Minecraft NeoForge.

### Requirements

- Java 21
- Minecraft 1.21.1
- NeoForge 1.21.1
- Gradle / Gradle Wrapper included in the project

### How to build

Clone the project:

```bash
git clone <repository-url>
cd <project-folder>

Build the mod:

./gradlew build

On Windows:

gradlew.bat build

The compiled .jar file will be generated in:

build/libs/
Development setup

For IntelliJ IDEA:

./gradlew genIntellijRuns

On Windows:

gradlew.bat genIntellijRuns

Then open the project in IntelliJ IDEA and run the Minecraft client configuration.
